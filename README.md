# Predictive_Analytics_Advance_Mathematics

## 1. Methodology
Data Collection → Column Selection → Data Transformation → Parameter & Probability Function Estimation → Parameter Printing  

## 2. Description
- **Notebook:** `Predictive_Analytics_Advance_Mathematics.ipynb`  
- **Dataset:** `data.csv`  
- **Target Column:** `no2`  

### Transformation Applied
z = Tr(x) = x + asin(bx)

### Probability Function Used
p(z) = c * e ^ (-lamda * (z - mu) ^ 2)

## 3. Objectives
- Transform the dataset mathematically  
- Estimate distribution parameters  
- Apply parameter estimation techniques  
- Print optimal values of parameters (λ, μ, c)  

## 4. Input
- CSV dataset (`data.csv`)  
- Numerical features with target variable `no2`  

## 5. Output
- Transformed data  
- Estimated parameters (λ, μ, c)  
- Printed optimal parameter values
